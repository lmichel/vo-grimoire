Ingestion process:

  • Mail archives download from the IVOA site
  • JSON serialization of the messages
  • formatting and en/decoding of the JSON elements to be readable by Elastic Search.
  • JSON strings indexing by Elastic Search.

The whole operation takes about 6 minutes

The Web application is simple ElasticSearch frontend with an advanced layout based on BootStrap.
